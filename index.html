HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Make sure to check index.html in VS Code for review of Study Guide webpage.

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • You can use hexadecimal code to define color, which you can veiw on HTML Color Codes website
  • Border, margin, and box-shadow are all elements that can be assigned usingpx markers.
  • Html files must be linked with css style sheets for visual updates to be applied on a browser
  • CSS Rule: selector defines the element or attributes to which the rules or declarations will apply
  • Declarations contain properties and values with a property: value declaration
  • Multiple declarations are contained in a declaration block beginning and ending with squiggly brackets

GIT

  • git status: checks what branch we are currently on.
  • git checkout-b branch-name: creates a new branch and switches to it.
  • Pull request merges a comparison branch to the main branch *leave comments*
  • git add -A adds modifications in the current working branch to the staging area.
  • git commit -m commits the current staged changes.
  • git pull receives github branch modifications to the local device.
  • git push pushes changes from our local device to github.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Functions must be called upon to execute the command.
  • We can use conditional statements and loops to control the order in which code is executed.
  • Array is a single variable that stores grouped data.
  • Index in code start with 0.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
